# Guía de Diseño y Formato: Lead Magnet PDF

Esta guía te ayudará a darle un aspecto profesional y coherente con tu marca (alejandro.dev) al PDF de los "7 Errores Técnicos".

## 1. Identidad Visual (Branding)
Para que el PDF parezca una extensión de tu web, usa estos códigos:
*   **Color Primario:** `#ffb800` (Amarillo vibrante - Úsalo para resaltar ideas, iconos y botones).
*   **Color Secundario:** `#2d3436` (Gris oscuro - Úsalo para el texto principal y fondos de contraste).
*   **Tipografía:** 
    *   **Títulos:** *Outfit* o *Montserrat* (Fuentes modernas y geométricas).
    *   **Cuerpo:** *Inter* o *Open Sans* (Alta legibilidad).

---

## 2. Formato por Páginas

### Página 1: Portada
*   **Fondo:** Color oscuro (`#2d3436`) con un degradado sutil hacia negro.
*   **Texto:** Título en blanco y amarillo (`#ffb800`) para las palabras clave "7 Errores Técnicos".
*   **Elemento Visual:** Una imagen de fondo abstracta de código o una foto profesional tuya con un filtro oscuro.

### Página 2: Introducción
*   **Layout:** Mucho espacio en blanco. Texto centrado.
*   **Elemento Visual:** Una gráfica simple que muestre "Éxito vs Fracaso" o simplemente una cita grande con el texto de la introducción.

### Páginas 3 a 9: Los 7 Errores (Estructura Repetitiva)
Para que sea fácil de leer, mantén el mismo diseño en estas 7 páginas:
1.  **Cabecera:** "Error #X" en una caja amarilla en la esquina superior.
2.  **Título:** En grande y negrita (Color oscuro).
3.  **Cajas de Contenido:**
    *   **Caja ROJA sutil:** Para "El Problema".
    *   **Caja VERDE/AMARILLA sutil:** Para "La Solución".
4.  **Elemento Visual Sugerido:**
    *   *Error 1:* Captura de pantalla de Google PageSpeed en rojo.
    *   *Error 2:* Icono de una base de datos con un candado roto.
    *   *Error 3:* Un dibujo de una web llena de parches (efecto Frankenstein).
    *   *Error 4:* Icono de un servidor echando humo o un aviso de "Error 500".
    *   *Error 5:* Icono de una carpeta abierta con billetes saliendo (pérdida de valor).
    *   *Error 6:* Foto de una pila de facturas desordenadas vs una interfaz de Stripe limpia.
    *   *Error 7:* Un reloj de arena agotándose.

### Página 10: Llamada a la Acción (CTA)
*   **Fondo:** Color amarillo (`#ffb800`) para generar urgencia y atención.
*   **Texto:** Color oscuro (`#2d3436`) para que contraste bien.
*   **Botón:** Haz que parezca un botón real (con sombra) aunque sea un link de PDF. Pon el texto "AGENDAR MI DIAGNÓSTICO GRATIS" en blanco.

---

## 3. Consejos Finales
*   **Exportación:** Guarda siempre como "PDF para impresión" o "PDF de alta calidad" en Canva para que los textos no se vean borrosos.
*   **Enlaces:** ¡Importante! Asegúrate de que el botón final tenga el enlace a tu Calendly configurado.
*   **Menos es más:** No llenes las páginas de texto. Deja que el diseño "respire". Si el cliente ve un bloque gigante de texto, no lo leerá.
